# Troubleshooting

## Common Issues

### Job Stuck in Queue
- Check device availability
- Review queue depth
- Consider alternative device

### Low Result Quality
- Increase shot count
- Check circuit depth
- Use error mitigation

### High Costs
- Use simulators for testing
- Optimize circuit design
- Reduce shot count where possible

### Device Unavailable
- Check maintenance windows
- Use alternative devices
- Fall back to simulators
